# 実行方法

レポジトリをダウンロード
```
git clone https://github.com/lightondust/finteract_streamlit.git
```

使いたいブランチをチェックアウト
```
git checkout <branch>
```

ライブラリインストール
```
pip install -r requirements.txt
```

streamlitアプリを実行
```
streamlit run streamlit_app.py
```
